## technical notes

- No build process required — edit `index.html` directly
- Dropdown menus reposition dynamically via JavaScript on load and resize
- Fully responsive with a mobile overlay layout at ≤ 680px width
- Font: [Nerko One](https://fonts.google.com/specimen/Nerko+One) via Google Fonts
